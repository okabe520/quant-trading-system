# 量化多因子策略回测系统 v1.1

基于 Dash 的 A 股量化多因子策略回测与模拟投资跟踪系统。

## 功能

- **日频回测引擎** — 含手续费、滑点、印花税模拟
- **12 因子模型** — 动量(3) / 波动率(2) / 量比(2) / RSI / MA偏离 / 布林位置 / MACD / 短期反转
- **周度调仓** — 5 日调仓周期，降低交易成本
- **可视化仪表盘** — 权益曲线、回撤分析、持仓热力图、因子得分、交易记录
- **当日操作建议** — 基于最新数据生成推荐持仓，含调仓周期提醒
- **一键模拟投资** — 点击执行策略，自动结算上轮盈亏 + 建仓新一轮
- **投资历史常驻** — 页面启动即显示，累计收益/胜率/每轮 P&L 一览
- **自动加载** — 打开页面自动联网拉取全套数据并完成回测
- **缓存加速** — 本地 CSV 缓存，秒级回测

## 快速开始

```bash
pip install -r requirements.txt
python main.py
```

浏览器打开 http://localhost:8050，自动加载数据并展示回测结果。

## 使用方式

| 按钮 | 说明 |
|------|------|
| **缓存加载** | 仅用本地缓存，秒级出结果 |
| **联网回测** | 联网拉取最新数据后回测 |
| **一键模拟投资** | 卖出上轮持仓（结算盈亏），买入当日推荐（等权建仓） |

页面启动时自动执行一次联网回测，之后可调整参数（股票池、日期、持仓数）重新运算。

## 项目结构

```
├── main.py          # 入口
├── dashboard.py     # Dash Web 仪表盘（含投资跟踪）
├── backtest.py      # 回测引擎
├── strategy.py      # 因子合成与信号生成
├── factors.py       # 因子计算（12因子）
├── data.py          # 数据获取与缓存
├── config.py        # 配置参数
└── cache/           # 数据缓存 + 投资历史
```
