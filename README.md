# 量化多因子策略回测系统 v1.0

基于 Dash 的 A 股量化多因子策略回测与操作建议系统。

## 功能

- **日频回测引擎** — 含手续费、滑点、印花税模拟
- **10 因子模型** — 动量(3) / 波动率(2) / 量比(2) / RSI / MA偏离 / 布林位置
- **可视化仪表盘** — 权益曲线、回撤分析、持仓热力图、因子得分、交易记录
- **当日操作建议** — 基于最新数据生成推荐持仓，次日自动跟踪收益
- **缓存加速** — 本地 CSV 缓存，秒级回测

## 快速开始

```bash
pip install -r requirements.txt
python main.py
```

浏览器打开 http://localhost:8050

## 使用方式

| 按钮 | 说明 |
|------|------|
| **缓存加载** | 仅用本地缓存，秒级出结果 |
| **联网回测** | 联网拉取最新数据后回测 |
