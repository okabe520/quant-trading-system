1 策略总览

一句话概括

本系统是一个基于A股日频数据的量化多因子选股回测平台。在每个交易日，对股票池内所有股票的10个原始因子进行截面z-score标准化，按预设权重加权求和形成综合得分，选出得分最高的N只股票等权重配置，每日调仓，纯多头交易。

2 因子体系详解

本策略共使用10个原始因子，覆盖六大类别。每个因子在每日截面上独立计算，对所有股票进行 z-score 标准化后按权重合成。以下逐一解释每个因子的经济含义和计算方式。

类别因子名称回看期原始权重权重方向

动量momentum_1010 天+0.10做多

2.1 动量因子（Momentum）

经济学逻辑

动量效应是学术界验证最稳健的市场异象之一——过去表现优异的股票倾向于在未来继续跑赢。Jegadeesh & Titman (1993) 首次系统证明了 3~12 个月动量策略在美国市场显著有效。在A股市场，由于交易者结构偏散户、信息扩散较慢，中期动量（1~3 个月）更为显著，而短期反转效应也存在。

本策略使用多周期动量（5/10/20日）的加权组合，形成动量强度频谱——既捕捉短线趋势，也捕捉中线惯性。20日动量权重最高（+0.15），体现了对中期趋势的侧重。

momentum_N = (当日收盘价 - N日前收盘价) / N日前收盘价
= close.pct_change(N)

2.2 波动率因子（Volatility）— 负权重

经济学逻辑

低波动率异象（Low-Volatility Anomaly）是继CAPM之后最重要的实证发现之一。Ang et al. (2006) 发现高特质波动率股票的后续收益反而更低。在A股市场，这一效应同样显著——散户投资者普遍追逐高波动"彩票型"股票，导致其定价过高；而机构偏好的低波动蓝筹股反而获得超额收益。

本策略对波动率因子赋予负权重（-0.10 和 -0.05），意味着系统偏好低波动率的股票。这是通过负权重实现的——波动率越高的股票 z-score 越大，乘以负权重后将压低综合得分，从而降低被选中的概率。

volatility_N = 日收益率滚动N日标准差 × √252
（年化波动率）

2.3 成交量比率（Volume Ratio）

经济学逻辑

成交量是市场关注度和资金流向的代理变量。量比大于1意味着当日成交活跃度超过近期均值，通常伴随信息冲击或主力资金介入。本策略将量比视为正向因子——放量上涨的股票往往具有更强的短期动能。两个周期（5日/20日）分别捕捉短期和中期资金关注度变化。

volume_ratio_N = 当日成交量 / N日均成交量

2.4 相对强弱指标（RSI）

经济学逻辑

RSI是衡量价格变化速度的动量振荡器。传统用法中，RSI > 70 为超买（应卖出），RSI < 30 为超卖（应买入）。但本策略将RSI作为正权重趋势因子——RSI越高越偏好——这与动量因子的方向一致。策略预期处于上升趋势中的股票即使RSI偏高，仍可持续上涨（趋势延续而非反转）。

RSI = 100 - 100 / (1 + RS)
其中 RS = N日内平均涨幅 / N日内平均跌幅

2.5 均线偏离（MA Deviation）

经济学逻辑

价格相对于移动平均线的偏离度，衡量了短期价格与中期趋势基准的关系。正偏离（价格 > MA）表示股价处于上升通道，趋势偏强。完全基于均值回复的策略会做空正偏离，但本策略结合了趋势和均值回复，仅给予+0.10的温和正权重。

MA_deviation_20 = (收盘价 - MA20) / MA20

2.6 布林带位置（Bollinger Band Position）

经济学逻辑

布林带位置衡量股价在布林带内的相对位置（0=下轨，0.5=中轨，1=上轨）。这是本策略中权重最高的单因子（+0.15）。正值权重意味着偏好在布林带上轨附近的股票——这些股票处于强势区，通常是趋势加速启动的阶段。这与"强势股恒强"的逻辑一致。

BB_upper = MA20 + 2 × 标准差
BB_lower = MA20 - 2 × 标准差
BB_position = (收盘价 - BB_lower) / (BB_upper - BB_lower)
理论取值范围 [0, 1]

因子权重分配逻辑

权重总和 = 1.0（正权重 0.85 + |负权重| 0.15）
动量类总权重：0.35（momentum_5: 0.10 + momentum_10: 0.10 + momentum_20: 0.15）
低波类总权重：0.15（volatility_10: -0.10 + volatility_20: -0.05）
量价类总权重：0.15（volume_ratio 系列）
技术类总权重：0.35（RSI + MA偏离 + 布林带位置）
⇒ 策略核心 = 动量（35%）+ 趋势强度（35%）+ 低波动率（15%）+ 流动性（15%）

3 因子合成与信号生成机制

3.1 截面 z-score 标准化

对于每个交易日 t，对每个因子的N只股票取值做截面标准化：

z-score(i,f,t) = (value(i,f,t) - mean(f,t)) / std(f,t)

为什么要截面标准化？

  • 不同因子的量纲不同（收益率是百分比，波动率是百分比，量比是无量纲比值），截面z-score使它们在同一尺度上可比。
  • z-score 本质上是"该股票在该因子上比同行强/弱了几个标准差"——衡量相对强弱而非绝对水平。
  • 如果某日所有股票的某一因子值完全相同，std = 0，则 z-score 设为 0（该因子该日不提供信息）。

3.2 Winsorize 截尾处理

标准化后的因子值被截断至 [-3, 3]：

z* = clip(z, -3.0, 3.0)

这防止了个别股票因极端因子值而过度主导综合得分。3σ 以外的值被认为是"噪声"而非有效信号。

3.3 综合得分计算

对每只股票 i 在交易日 t，综合得分为所有因子 z-score 的加权和：

Score(i,t) = Σ wf × z*(i,f,t)
= 0.10×z_mom5 + 0.10×z_mom10 + 0.15×z_mom20
  - 0.10×z_vol10 - 0.05×z_vol20
  + 0.10×z_volratio5 + 0.05×z_volratio20
  + 0.10×z_rsi + 0.10×z_madev + 0.15×z_bbpos

某个因子得分较高意味着：

  • 动量因子高 → 近期涨幅大 → 综合得分 ↑
  • 波动率因子高 → 波动大 → 综合得分 ↓（因为权重为负）
  • 量比高 → 放量 → 综合得分 ↑
  • RSI高、价格在布林带上轨 → 强势 → 综合得分 ↑

3.4 信号生成与选股

每个交易日，选股逻辑为：

  1. 将所有股票按综合得分从高到低排序
  2. 选取得分最高的 max_positions（默认8）只股票
  3. 附加条件：仅买入得分 > 0 的股票（得分 ≤ 0 的股票即使排名靠前也不会买入）
  4. 信号值 = 1（买入/持有），否则 = 0
重要特性：实际持仓数量并非固定。如果少于N只股票得分 > 0，则当日只持有这些股票，可能低至0只（空仓）。这提供了隐含的市场下行保护——当整体市场走弱时，多数股票因子得分转负，系统自动减仓甚至空仓。

3.5 目标权重分配

对选出的K只股票，采用最简单的分配方式：

wi = 1 / K   （等权重）

没有使用风险平价、波动率倒数加权、最小方差等更复杂的组合优化方法。这保持了策略的透明性和可解释性，但也意味着如果某只持仓股波动率极高，它将与波动率最低的持仓股承担相同的权重。

4 回测引擎设计

4.1 日频模拟交易循环

回测引擎按交易日逐日遍历，执行以下步骤：

  1. 估值：按当日收盘价计算当前组合总市值（持仓市值 + 现金）
  2. 获取目标：读取该日目标权重矩阵
  3. 卖出：计算每只持仓股需卖出的数量，先卖以释放现金
  4. 买入：用可用现金（最多 99%）按目标权重买入
  5. 记录：保存当日持仓、现金和总权益

4.2 交易约束与成本模型

约束/成本规则说明
整手交易 买入/卖出股数向下取整至 100 股的整数倍 A股最小交易单位为 1 手 = 100 股，不可买卖零股
滑点 卖出价 = 收盘价 × (1 - 0.01%)
买入价 = 收盘价 × (1 + 0.01%)		 模拟市场冲击成本和买卖价差（bid-ask spread）
佣金 成交金额 × 0.03%（万三），买卖双向 行业标准费率
印花税 卖出金额 × 0.1%（千一），仅卖出 A股法定印花税，2023年8月后降至千一
现金约束 买入金额 ≤ 可用现金 × 0.99 预留 1% 缓冲，防止因取整和价格波动导致的超买
已知局限（前瞻偏差）：回测在每日调仓时使用当日收盘价计算并成交，这在实盘中不可实现（当日收盘价在收盘后才确定，无法在盘中以收盘价下单）。这意味着回测收益可能被高估。实盘替代方案应使用次日开盘价或VWAP执行。

6 策略核心理念总结

策略的四维逻辑

趋势跟踪 (0.35)

买入近期涨幅最大的股票。认为上涨趋势具有持续性——"让赢家继续跑"（Let winners run）。三重动量周期覆盖短期到中期。

低波动率溢价 (0.15)

回避高波动率的"彩票型"股票，偏爱低波动稳健品种。利用市场对高波动股票的过度定价套利。

资金关注度 (0.15)

量比为信号捕捉市场关注度的变化。放量意味着信息冲击和资金介入——"量在价先"。

强势确认 (0.35)

RSI + 均线偏离 + 布林带位置三重确认趋势强度。偏好站上均线、布林带上轨附近的"强势股"。

隐含风控机制：综合得分 > 0 的过滤条件 + 截面排名机制，使得在市场普跌时自动降低仓位。等权配置避免了单只股票的过度集中风险。每日调仓确保组合始终持有当前最强因子得分的股票。

策略优劣势分析

优势局限与改进方向
✅ 多因子分散化——不依赖单一信号源 ⚠️ 仅做多——下跌市只能减仓无法对冲
✅ 低波动率因子提供防御性 ⚠️ 前瞻偏差——使用当日收盘价调仓
✅ 每日调仓适应快速市场变化 ⚠️ 生存偏差——股票池固定为当前大盘股
✅ 等权分配简单透明 ⚠️ 无止损/止盈风险管理机制
✅ 免费数据源（BaoStock/AKShare），零成本运行 ⚠️ 等权忽略了个股波动率差异
✅ Dash仪表盘——可视化丰富、参数可调 ⚠️ 未利用MACD等已实现但未启用的因子
